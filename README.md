## LeetCode026: 删除排序数组中的重复项      
https://leetcode-cn.com/problems/remove-duplicates-from-sorted-array/ <br>
解题思路：删除重复项。从后开始遍历。记住最后一项然后对比前项和后项数据，相同删除，不同重新记录。
C++容器可以自行改变大小并进行缩进，较为容易实现。
